# 파일 00 — 헤더 + 푸터 (사이트 공통)

## 1. 헤더 (Header)

### PC 헤더
- 로고: **몽젤로** 단일, 좌측 고정
- GNB(좌측~중앙): `몽젤로` · `젤로베베` · `브랜드 스토리` · `룩북` · `리뷰` · `고객센터`
- 우측 유틸리티: **장바구니(SVG)** · **마이페이지(SVG)** — 아이콘 2개만
- 브랜드 토글 없음. GNB에 전화번호/카톡 아이콘 노출 안 함.

**[레이아웃]** 3분할 헤더: `로고(좌)` — `GNB 가로 메뉴(중앙)` — `아이콘 2개(우)`. 높이 64~72px. `몽젤로`·`젤로베베`는 2depth 드롭다운(카테고리 펼침), 나머지는 단일 링크. 아이콘은 stroke 기반 라인 아이콘(장바구니=shopping-bag, 마이페이지=user). 장바구니 아이콘에 수량 배지(badge) 슬롯.

### 모바일 헤더
- 로고: 좌측 / 우측: 장바구니(SVG) · 마이페이지(SVG) · 햄버거
- GNB는 햄버거 → 좌측 드로어(Drawer)로 펼침. 드로어 안에서 `몽젤로`/`젤로베베`는 아코디언으로 하위 카테고리 노출.
- 하단 고정 바 없음.

**[레이아웃]** 상단 고정 56px. 햄버거 탭 시 좌측 슬라이드 드로어(85% 폭) + 딤드 오버레이.

### 스크롤 동작
- PC: 스크롤 다운 시 헤더 축소 고정(로고 약간 작아지고 GNB 유지), 반투명→불투명.
- 모바일: 상단으로 스크롤 시 헤더 재노출(hide on scroll-down / show on scroll-up).

# 파일 01 — 메인 (통합 브랜드 허브)

📋 유형: P-MAIN
📌 목적: 두 브랜드를 한 화면에서 직관적으로 분기시키고, 시그니처(카이만)와 소재 신뢰로 첫 방문자를 상품 탐색에 진입시킨다.
👤 방문자: 광고/인스타/검색 유입. 브랜드를 처음 접하거나, 몽젤로는 알지만 젤로베베는 모르는 기존 고객.
🎯 전환 액션: 브랜드 선택 → 카테고리 진입 → 상품 탐색
🔗 유입 경로: 인스타 프로필 링크, 검색, 광고, 직접 방문

──────────────────────────────────────
섹션 1: 히어로 (3슬라이드 자동 전환) — 고정 포맷
──────────────────────────────────────
- 목적: 3초 내 "어떤 브랜드인지 + 어디로 가면 되는지" 인지
- 슬라이드 1 (핵심 가치): 헤드 "오래 입혀도 질리지 않는 옷" / 서브 "유행이 아니라, 아이의 시간에 맞춘 디자인"
- 슬라이드 2 (시그니처·신뢰): 헤드 "한 장의 사진으로 시작된 브랜드" / 서브 "카이만 패딩, 그 겨울의 따뜻함을 기억합니다" [자동적용 — 판매수치 카피는 확정 후 삽입]
- 슬라이드 3 (전환 유도): 헤드 "영유아부터 키즈까지, 한 곳에서" / 서브 "젤로베베와 몽젤로를 지금 만나보세요"
- 전환 속도: 4~5초 자동 + 도트 네비

**[레이아웃]** 풀스크린(100vw × 80~90vh) 이미지 배경 캐러셀. 텍스트 좌하단 또는 중앙 정렬. 하단 중앙 도트 인디케이터. 모바일은 4:5 비율로 축소.

──────────────────────────────────────
섹션 2: 두 브랜드 분기 (듀얼 진입)
──────────────────────────────────────
- 목적: 메인의 핵심 기능 — 방문자를 자기 자녀 연령 브랜드로 즉시 분기
- 헤드카피: "우리 아이는 지금 어느 계절인가요"
- 콘텐츠: 좌/우 2분할 카드. 각 카드 = 브랜드 무드컷 + 한 줄 설명 + 진입 버튼
  · 몽젤로: "자기 취향이 생긴 아이를 위한, 시크하고 단정한 키즈룩" → [몽젤로 보기]
  · 젤로베베: "여린 피부에 닿는 첫 옷, 정직한 소재의 영유아룩" → [젤로베베 보기]

**[레이아웃]** PC 좌우 50:50 풀블리드 2분할(hover 시 확대/딤). 모바일 상하 2단 스택. 각 카드 클릭 전체가 링크.

──────────────────────────────────────
섹션 3: 시그니처 — 카이만 패딩
──────────────────────────────────────
- 목적: 브랜드를 각인시킨 히트 아이템으로 신뢰·호기심 유발
- 헤드카피: "겨울마다 다시 찾는 이유가 있어요"
- 서브카피: "가볍게, 그러나 충분히 따뜻하게 — 카이만 오리털 라인"
- 콘텐츠: 카이만 대표컷 + 짧은 소재/핏 포인트 3개 + [카이만 보러가기]

**[레이아웃]** 좌측 대형 이미지 / 우측 텍스트 + 포인트 리스트(아이콘 3). 모바일 상하. 배경 톤 다르게 구획.

──────────────────────────────────────
섹션 4: 브랜드별 베스트
──────────────────────────────────────
- 목적: 실제 상품으로 진입시키는 1차 상품 노출
- 헤드카피: "지금 가장 많이 담는 옷"
- 콘텐츠: 탭(몽젤로 / 젤로베베) 전환형 상품 그리드. 각 탭당 상품카드 4~8개. 카드 = 썸네일·상품명·가격·별점·[오가닉 라벨(해당 시)]

**[레이아웃]** 상단 탭 토글 → 하단 상품 그리드(PC 4열 / 모바일 2열). 카드 hover 시 2nd 이미지. "더 보기" → 해당 브랜드 카테고리.

──────────────────────────────────────
섹션 5: 소재·진정성 스토리 티저
──────────────────────────────────────
- 목적: 소재 신뢰 부재(핵심 페인포인트) 선제 해소, 스토리 페이지로 유도
- 헤드카피: "보이지 않는 곳까지, 직접 봅니다"
- 서브카피: "어디서 어떻게 만드는지 떳떳하게 보여드릴 수 있는 옷"
- 콘텐츠: 무드 이미지(제작/원단 디테일) + 1~2문장 + [브랜드 스토리 보기]

**[레이아웃]** 풀블리드 이미지 위 텍스트 오버레이 또는 좌이미지 우텍스트. 절제된 저밀도 레이아웃.

──────────────────────────────────────
섹션 6: 실시간 포토 리뷰
──────────────────────────────────────
- 목적: 사회적 증거 — '나 같은 엄마' 후기로 구매 확신
- 헤드카피: "엄마들이 직접 남긴 후기"
- 콘텐츠: 포토 리뷰 가로 슬라이드(이미지+별점+한줄). [리뷰 전체 보기]

**[레이아웃]** 가로 스크롤 카드 캐러셀(PC 4~5개 노출 / 모바일 1.5개 peek). 이미지 정사각.

──────────────────────────────────────
